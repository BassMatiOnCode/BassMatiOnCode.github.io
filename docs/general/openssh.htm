OpenSSH

Manual Installation

  1. Download https://github.com/PowerShell/Win32-OpenSSH/releases and extract the OpenSSH-Win64 folder to the desired destination.
  2. Grant read and execute permissions to all users on the libcrypto.dll
  3. Run install-sshd.ps1 to create the services OpenSSH Authentication Agent and OpenSSH Server, set permissions and register an event tracing (ETW) provider.
  4. Set the startup type and start the server: 
    &sc.exe config ssht start= auto
&sc.exe config ssh-agent start= auto
&sc.exe start ssh-agen
  5. Open port 22 in the Windows Defender Firewall by enabling the rule OpenSSH-Server-In-TCP, or create a rule manually: 
    New-NetFirewallRule -Name sshd -DisplayName 'OpenSSH SSH Server' -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22 -Program "C:\Windows\System32\OpenSSH\sshd.exe"
    Note to adjust the path to your needs.
