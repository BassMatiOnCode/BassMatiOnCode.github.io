RSync in GIT Bash

  1. Download the following files to a temporary folder:
    • rsync-3.2.3-2-x86_64.pkg.tar.zst
    • libxxhash-0.8.1-1-x86_64.pkg.tar.zst
    • liblz4-1.9.3-1-x86_64.pkg.tar.zst
    • libzstd-1.5.2-1-x86_64.pkg.tar.zst
  2. Download and extract zstd-v1.5.2-win64.zip to a tool folder that is in your path environment variable.
  3. Start GIT bash and navigate to the download folder.
  4. Then execute the following commands: 
    zstd rsync-3.2.3-2-x86_64.pkg.tar.zst
	zstd libxxhash-0.8.1-1-x86_64.pkg.tar.zst
	zstd liblz4-1.9.3-1-x86_64.pkg.tar.zst
	zstd libzstd-1.5.2-1-x86_64.pkg.tar.zst
  5. Extract the tar archives with the following commands: 
    tar -xf rsync-3.2.3-2-x86_64.pkg.tar.zst
	tar -xf libxxhash-0.8.1-1-x86_64.pkg.tar.zst
	tar -xf liblz4-1.9.3-1-x86_64.pkg.tar.zst
	tar -xf libzstd-1.5.2-1-x86_64.pkg.tar.zst
    This will create a usr/ folder with the subfolders bin/, lib/ and share/.
  6. Move these subfolders to C:\Program Files\Git\usr\. Click OK to the merge and permission dialogs in case they appear.
  7. Still in Git Bash, run the following command: 
    rsync
    If everything is ok, a help screen should show up.
